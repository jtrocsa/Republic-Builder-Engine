# Milestone 3.1 — Chronicle Institute Archive Loop

This milestone establishes the reusable high-level loop for Chronicle:

1. Enter the present-day Chronicle Institute.
2. Enter the Archive.
3. Use the map table to choose a teacher-unlocked historical route.
4. Chronotravel to a local field station.
5. Read actual source excerpts inside the game, secure them in the Codex, and submit a field record.
6. Return to the Chronicle Institute.

## Important boundaries

- The **Chronicle Institute** is the permanent present-day home base.
- A historical setting may have a **Field Station**. It is a local safe point, not the main Archive.
- Source readers must show the actual source text or visual inside the experience. An archive link is supplementary.
- The current sources are a compact prototype source set. Future cases add their own source records using the same structure.
