# Milestone 3 — Case 1.01 Field Experience

This milestone introduces the first playable investigation loop for **Case 1.01: The Atlantic Crossroads**.

## Player flow

1. Speak with Maren Vale at the field station.
2. Open the Case Briefing.
3. Enter a 1492 Caribbean field setting.
4. Walk to three evidence signals.
5. Read each record in the Codex and secure it.
6. Complete one sourcing/point-of-view check.
7. Review a case recap showing Observe → Source → Report.

## Scope boundary

This is a playable vertical-slice preview. It does not yet include the full Exchange Ledger, MCQ set, SAQ workspace, teacher assignment mode, or permanent student progress tracking.

## Content boundary

The case separates contextual information about Taíno societies from a European primary-source account and a visual-source warning. That distinction is intentional: students practice identifying what a source can show and what it cannot show by itself.
