# Milestone 3.3 — Unit 1 Campaign Expansion

This milestone turns Unit 1 into a playable mini-campaign built around a stable campaign loop:

`Institute → Archive map → Chronotravel → Case activity → Codex → Case completion → Archive → Unit review`

## Map rules

- The Archive table is a present-day navigation instrument, so it remains parchment/brass/scholarly rather than pixel art.
- Historical field settings are top-down pixel-art exploration spaces.
- The chronotravel sequence visually connects both styles: map marker → route line → geographical zoom → historical field.
- Unit maps show only locations associated with the active unit. Do not retain later-unit locations merely as decoration.

## Unit 1 case structure

| Case                               | Place                      | Signature activity      | Unlock condition                                                                |
| ---------------------------------- | -------------------------- | ----------------------- | ------------------------------------------------------------------------------- |
| Case 1.01: The Atlantic Crossroads | Caribbean, 1493            | Record Reconstruction   | Secure sources and correctly place three records in evidentiary lanes           |
| Case 1.02: The Exchange Ledger     | Atlantic routes, 1493–1540 | Atlantic Route Puzzle   | Correctly connect origin, destination, and consequence for four records         |
| Case 1.03: Empire's Foundations    | Spanish Caribbean, c. 1502 | Colonial System Builder | Correctly build five causal links and provide a short evidence-based reflection |
| Unit 1 Archive Review              | Chronicle Institute        | MCQ + SAQ               | Complete all prior cases                                                        |

## Assessment integrity

The mini-games prepare students for historical analysis; they do not replace it. Every case ends with an evidence-based claim, source response, or written reflection.
