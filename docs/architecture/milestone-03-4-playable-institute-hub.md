# Milestone 3.4 — Playable Chronicle Institute Hub

## Scope

- Replaces the static Institute landing card with a top-down foyer and Archive room.
- Uses the established PNG player sprite system; no generic CSS player art is introduced.
- Adds Director Rowan Hale, Dr. Amani Soto, and Professor Julian Park as recurring Institute NPCs.
- Makes the physical Navigation Table the canonical entry point to the geographic campaign map.
- Returns completed cases to the playable Institute, not directly to a menu.

## Controls

- Arrow keys / WASD: move inside the Institute.
- E / Enter: interact with the nearby person or Navigation Table.
- Click: accessibility alternative for people and table.

## Non-goals

- This milestone does not replace the Caribbean field map with the later living-historical-world rebuild.
- This milestone does not change Unit 1 assessment scoring.
