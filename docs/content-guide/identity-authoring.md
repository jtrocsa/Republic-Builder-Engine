# Chronicle Identity — Authoring Guide

## Editable through Author Mode

- Identity screen title, directions, labels, and button text
- Registration screen title, Codex description, assignment label, and button text
- Field-entry location title, mission directions, mentor name/role/dialogue, and completion labels

## Protected by design

Author Mode cannot change:

- The two appearance choices or sprite files
- Player movement rules
- Map tiles / blocked terrain
- Mentor interaction distance
- Profile storage shape
- Navigation sequence
- Game mechanics, scoring, or security

## Draft / publish workflow

1. Turn on Author Mode.
2. Revise copy and allow autosave to finish.
3. Export JSON when a draft is ready to back up or publish.
4. Replace the corresponding repository default content file only after approving the language.
5. Commit that intentional content update to GitHub.

The browser-local draft is for experimentation. The committed default file is the classroom-ready source of truth.
