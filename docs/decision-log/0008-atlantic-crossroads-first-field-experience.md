# Decision 0008 — First Field Experience

Case 1.01 begins with a constrained evidence-gathering loop instead of a large open-world map.

## Decision

The first playable case uses three evidence signals, each tied to a distinct historical-thinking move:

- Contextualize Indigenous Caribbean societies before contact.
- Source a European expedition account.
- Evaluate a visual source's provenance.

## Why

The vertical slice must demonstrate movement, exploration, primary-source handling, Codex evidence collection, and AP-style sourcing without requiring the entire Unit 1 quest system first.
