# Decision 0014 — Field Engine and Living Caribbean Correction

The previous 3.4.1 archive did not modify the implementation. This replacement pack changes the actual source code.

## Implemented corrections

- Removes the field mentor from the historical Caribbean map.
- Makes the Chronicle Institute the only Archive/Chronicler home base.
- Expands Case 1.01 into a larger scrollable Caribbean world.
- Adds a washed-up Columbus-era ship visual with mast, sail, and cross marker.
- Adds a Taíno village with bohío-style houses, garden area, canoe, and NPCs.
- Adds a Spanish camp with NPCs, campfire, supplies, and source-adjacent investigation space.
- Spreads the three source signals across the larger environment.
- Replaces grid-jump field movement with smaller, faster movement increments.
- Adds an island land mask so the player cannot walk into open water.
- Cleans the Institute Navigation Table overlay so it reads less like a random rectangle.
