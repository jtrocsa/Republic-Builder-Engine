# Decision 0019 — Field movement, reset controls, and Archive display

Milestone 3.4.5 is a stability-focused hotfix after the 3.4.4 interaction pass.

## Changes

- Moved the default Caribbean spawn to an open, walkable village-side position.
- Added a Case 1.01 demo reset button so iteration can be tested without manually clearing browser storage.
- Simplified the Institute label to “Institute Archive.”
- Removed the Foyer/Archive map stickers from the playable hub.
- Added a first preserved-records case in the Institute as the beginning of a badge/display system.
- Reduced the floating NPC patrol look and shifted to small footstep motion until full NPC walk cycles are built.
