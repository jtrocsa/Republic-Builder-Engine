# Decision 0006 — Field Definition Pass

Milestone 2.2 improves the presentation layer of the first playable field scene without changing the underlying game architecture.

## Goals

- Increase immersion by expanding the visible play area.
- Reduce the sense of a boxed-in demo panel.
- Move from a rigid side panel toward a lighter overlay/HUD.
- Increase environmental readability so the scene reads as a place rather than a flat tile grid.
- Add subtle environmental motion.

## Non-goals

- No new assessment systems.
- No new unit content.
- No major architectural changes to movement/collision/content loading.
