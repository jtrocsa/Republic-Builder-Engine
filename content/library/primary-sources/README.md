# Primary Source Library

This is the one authoritative home for source metadata.

Do not copy a source into a case folder.

Each source receives:

- one metadata record (`source-...json`)
- optional transcription or excerpt (`source-...md`)
- a media path pointing into `assets/campaigns/<campaign>/documents/`

Cases reference source IDs from this library.

Use `source-record.template.json` as the starting point.
